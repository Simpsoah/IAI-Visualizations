Visualization Information

Nodes are sized by the number of tweets a user has made interacting with another user, either as a Retweeting, Replying to, or directly mentioning a user. The color of the nodes indicate the number of times the user has Retweeted other user's messages. Edge color represents a the number of times a source node has retweeted a target node's message.
